:::info

FlutterFire UI is still in beta and is subject to change. Please contribute to the [discussion](https://github.com/FirebaseExtended/flutterfire/discussions/6978) with feedback.

:::

## Getting Started

To install FlutterFire UI, ensure you have first [installed & initialized FlutterFire](https://firebase.flutter.dev/docs/overview) by following the
documentation.

Next, install the `flutterfire_ui` package by running the following command in your terminal:

```bash
flutter pub add flutterfire_ui
```
